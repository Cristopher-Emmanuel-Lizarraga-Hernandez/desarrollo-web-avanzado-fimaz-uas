# Práctica 1 - Clases y Encapsulamiento en PHP

## Objetivo
Aplicar los fundamentos de la Programación Orientada a Objetos en PHP mediante la creación de una clase que utilice encapsulamiento, constructor y métodos de acceso.

## Descripción de la clase

Se creó una clase llamada **Usuario** que contiene:

Atributos privados:
- nombre
- correo

Métodos implementados:
- Constructor para inicializar los datos
- Métodos Getter:
  - getNombre()
  - getCorreo()
- Métodos Setter:
  - setNombre()
  - setCorreo()

Esto permite proteger los datos del objeto y acceder a ellos mediante métodos controlados.

## Instrucciones de ejecución

1. Colocar la carpeta del proyecto dentro del servidor web (por ejemplo, htdocs en XAMPP).
2. Iniciar Apache.
3. Abrir el navegador y acceder a:


http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-1/


4. Se mostrará en pantalla la información del usuario creada desde la clase.
